# 🚀 DSA Practice in Java

A collection of **Data Structures and Algorithms (DSA)** programs implemented in **Java**. This repository is created for learning, interview preparation, coding practice, and improving problem-solving skills.

### 📁 Prefix Sum, Sliding Window, Two Pointer
Contains implementations and problems based on:
- Prefix Sum
